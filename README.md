# intern_deploy - Vercel + Render deployment-ready monorepo

This repo is optimized for deployment:
- Frontend (apps/web) -> Vercel
- Backend (apps/api) -> Vercel (serverless or node)
- Vanna AI (apps/vanna) -> Render (service)
- Database -> Render PostgreSQL (recommended)

Replace placeholders in .env.production files with your actual secrets before deploying.
